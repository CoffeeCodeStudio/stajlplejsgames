# StajlPlejs Game Zone

En spelmodul byggd för StajlPlejs — ett svenskt nostalgicommunity med 40 000 registrerade medlemmar.

## Spel
- Memory
- Snake
- Scribble (multiplayer)

## Funktioner
- Scoreboard med topplista
- Dynamiskt användarnamn via URL-parameter
- Designmatchad mot StajlPlejs visuella stil

## Integration
```html
<iframe src="https://stajlplejsgames.lovable.app?usr=Användarnamn" width="100%" height="800" frameborder="0" scrolling="yes"></iframe>
```

## Teknik
- Byggd med Lovable AI
- Hostad via Lovable
- Ingen förändring i StajlPlejs databas eller kod krävs
